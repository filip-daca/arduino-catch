# Arduino Shotty

Simple 2x16 LCD game for Arduino UNO.
This is a scratch of a char LCD display game engine.

### Features
- custom sprites
- dynamic custom character allocation
- analogue movement controls
- digital fire button
- led health display
- multiple sources of sound
